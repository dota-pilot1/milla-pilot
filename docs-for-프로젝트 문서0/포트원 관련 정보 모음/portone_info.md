### 채널 설정 (테스트 — 카카오페이 일반결제)

| 항목 | 값 | 비고 |
|---|---|---|
| PG Provider | `kakaopay` | |
| PG상점아이디 (CID) | `TC0ONETIME` | 카카오페이 공식 테스트 CID |
| 결제 모듈 | 간편결제 일반/정기결제 (V2) | |
| 수동 승인 | **사용 안함** | 즉시 승인 — "결제 후 미달 시 환불" 모델과 일치 |
| 과세구분 | 미설정 | 부가세 신고 참고용. 기부금 면세 여부는 세무 확인 후 설정 |

테스트 채널은 토스페이·스마일페이도 생성돼 있으나, **계약 완료된 카카오페이만 사용**한다.

## 웹훅

| 항목 | 값 |
|---|---|
| 수신 URL (개발) | `http://localhost:4301/api/webhooks/portone` |
| 수신 URL (운영) | `https://<도메인>/api/webhooks/portone` |
| 콘솔 등록 여부 | ☐ |

**서버 필수 처리** — 서명 검증 · 금액 대조(서버가 만든 PENDING 금액과 비교) · 멱등성(`paymentKey` UNIQUE, 이미 PAID면 무시)
