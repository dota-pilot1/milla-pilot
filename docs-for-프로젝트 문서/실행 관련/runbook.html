Backend 환경 변수

./gradlew bootRun은 donation-platform-server/.env 파일이 있으면 읽어서 환경 변수로 주입한다. 

JWT_SECRET=donation-platform-dev-secret-key-must-be-at-least-32-bytes-long
AWS_ACCESS_KEY_ID=YOUR_AWS_ACCESS_KEY_ID
AWS_SECRET_ACCESS_KEY=YOUR_AWS_SECRET_ACCESS_KEY
AWS_S3_BUCKET_NAME=your-bucket-name
AWS_S3_REGION=ap-northeast-2
AWS_S3_PREFIX=donation-platform
S3 값이 비어 있어도 기본 API 실행은 가능하지만, 업로드 presign 기능은 실제 AWS 설정이 필요하다.

Docker / PostgreSQL

cd /Users/terecal/milla-pilot/donation-platform-server

# 실행
docker compose up -d postgres

# 상태 확인
docker compose ps

# 로그 확인
docker compose logs -f postgres

# 중지
docker compose stop postgres

# 컨테이너 제거
docker compose down

# 볼륨까지 제거: 로컬 DB 데이터 삭제됨
docker compose down -v
docker compose down -v는 donation_platform_postgres_data 볼륨을 삭제하므로 로컬 DB 데이터가 사라진다.

Swagger / API 확인

목적 URL 또는 API 비고
Swagger UI http://localhost:4301/swagger-ui/index.html SecurityConfig에서 permitAll 처리됨.
OpenAPI JSON http://localhost:4301/v3/api-docs API 스키마 확인.
로그인 POST /api/auth/login JWT access/refresh token 발급.
현재 사용자 GET /api/auth/me Bearer token 필요.
메뉴 GET /api/menus 웹/Tauri 사이드바 메뉴 확인.
시설 GET /api/facilities, GET /api/facilities/{id}/items 후원하기 화면의 기본 데이터 후보.
후원 물품 POST/PATCH/DELETE /api/donation-items 관리자 앱에서 CRUD 연결 예정.

주요 API 그룹

그룹 Base Path 용도
Auth /api/auth 회원가입, 이메일 확인, 로그인, 토큰 갱신, 로그아웃, 내 정보.
Users /api/users 사용자 관리, 역할/활성화 상태 변경.
Menus /api/menus DB 기반 메뉴 조회 및 관리.
Facilities /api/facilities 시설 목록, 상세, 시설별 후원 물품 조회, 시설 CRUD.
Donation Items /api/donation-items 후원 물품 생성, 수정, 삭제, 상세 조회.
Roles / Permissions /api/roles, /api/permissions, /api/permission-categories RBAC 관리.
Site Settings /api/site-settings 사이트 설정 조회/저장.
Upload /api/upload/presign S3 presigned upload URL 발급.

자주 보는 문제

증상 확인할 것 대응
백엔드 시작 실패 PostgreSQL 컨테이너, 5436 포트, DB 계정. docker compose ps, docker compose logs -f postgres 확인.
웹에서 API 호출 실패 NEXT_PUBLIC_API_URL, 백엔드 4301 실행 여부. 백엔드 실행 후 브라우저 새로고침.
Tauri 403 / CORS 백엔드 Security/CORS 허용 origin, Tauri devUrl http://localhost:1422. 서버 CORS 설정과 preflight 응답 확인.
Swagger 인증 API 실패 Bearer token 입력 여부. 로그인 API로 토큰 발급 후 Swagger Authorize에 입력.
메뉴가 안 보임 MenuSeeder 실행 로그, GET /api/menus 응답. 서버 재시작 후 DB 메뉴 데이터와 visible 값을 확인.
