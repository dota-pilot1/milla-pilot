7. 주의 사항

이 문서에는 AWS Access Key, SSH private key, DB 비밀번호, 메일 비밀번호, JWT secret 같은 비밀값을 절대 저장하지 않는다. GitHub Actions Secret 이름과 운영 접속 주소만 문서화한다.

현재 배포 기준

main에 push되면 CI가 자동 실행되고, CI가 성공하면 Deploy가 자동 실행된다. 같은 브랜치에서 새 배포가 시작되면 이전 배포는 취소된다.
