## 동작 기준

- 이메일 기준으로 멱등 처리한다.
- 이미 같은 이메일이 있으면 새로 만들지 않는다.
- `ROLE_DONOR`가 없으면 시드를 건너뛴다.
- 비밀번호는 `PasswordEncoder`로 해시 저장한다.

## 로그인 화면 연동

웹 로그인 폼은 개발 환경에서만 시더 계정 빠른 입력 UI를 노출한다.

- 위치: 로그인 폼 상단
- 표시 계정: `donor01@test.com`, `donor02@test.com`, `donor10@test.com`
- 클릭 시 이메일과 비밀번호를 자동 입력한다.
- 운영 빌드에서는 노출하지 않는다.
